# 公式说明书

## 一、OZON 售价公式

### 1. 物流成本（Ship Cost）
```
物流成本 = 3 + 0.045 × 产品重量(克)
```

示例：
- 200g 产品：3 + 0.045 × 200 = **¥12.00**
- 1000g 产品：3 + 0.045 × 1000 = **¥48.00**

**可自定义公式**，例如：
- 超重商品：`5 + 0.08 * weight`
- 固定运费：`10`
- 轻量商品：`2 + 0.02 * weight`

### 2. 实际成本（Real Cost）
```
实际成本 = 产品成本 × (1 + 货损率)
```

作用：考虑运输、破损、退货等损失

示例（成本¥20，货损率20%）：
```
实际成本 = 20 × (1 + 0.20) = 20 × 1.2 = ¥24.00
```

### 3. 建议售价（Final Price）- ⭐ 核心公式

```
建议售价 = (物流成本 + 实际成本) ÷ (1 - 佣金 - 广告费 - 利润)
```

**这个公式保证了**：
- 扣除平台佣金后有足够利润
- 广告投放成本被覆盖
- 货损被充分补偿

详细背景：
```
售价 × (1 - 佣金 - 广告 - 利润) = 物流成本 + 实际成本

取值：
售价 = (物流成本 + 实际成本) ÷ (1 - 佣金 - 广告 - 利润)
```

### 4. 四舍五入售价（Rounded Price）
```
四舍五入售价 = CEILING(建议售价)
```

即：向上取整到最近的整数

示例：
- ¥58.33 → **¥59**
- ¥59.01 → **¥60**
- ¥60.00 → **¥60**

## 二、完整计算示例

**输入参数：**
- 产品成本：¥20.00
- 产品重量：200g
- 佣金：20%
- 广告费：10%
- 货损率：20%
- 利润：30%

**计算过程：**

```
第1步：计算物流成本
物流成本 = 3 + 0.045 × 200 = 3 + 9 = ¥12.00

第2步：计算实际成本
实际成本 = 20 × (1 + 0.20) = 20 × 1.2 = ¥24.00

第3步：计算费率总和
费率总和 = 1 - 0.20 - 0.10 - 0.30 = 0.40

第4步：计算建议售价
建议售价 = (12 + 24) ÷ 0.40 = 36 ÷ 0.40 = ¥90.00

第5步：四舍五入
四舍五入售价 = CEILING(90.00) = ¥90
```

**结果：建议在 OZON 上架此商品的售价为 ¥90**

## 三、费率详解

### 佣金（Commission）
- OZON 平台抽取费用，通常 15% - 25%
- 不同品类可能不同
- **默认值：20%**

### 广告费（Ads）
- 投入于 OZON 平台广告的成本占比
- 范围：0% - 50%（看推广力度）
- **默认值：10%**

### 货损率（Damage）
- 考虑破损、退货、丢失等损失
- 物流损失率通常 10% - 30%
- **默认值：20%**

### 利润（Profit）
- 最终想要的利润空间
- 范围：10% - 50%（看定位）
- **默认值：30%**

## 四、公式应用场景

### 场景 1：利润目标调整
**问题**：想要 40% 利润，但费率太高怎么办？

**解除方法**：将 `利润` 改为 40%，重新计算会直接看到新售价。

### 场景 2：运费很高
**问题**：重型产品运费 ¥50，怎么定价？

**做法**：
1. 修改物流公式：`50` 或更精确的公式
2. 重新计算
3. 看新的售价建议

### 场景 3：薄利多销 VS 高利润小量
**对比 1** - 薄利多销（利润 15%）：
```BuildConfiguration
建议售价 = (12 + 24) ÷ (1 - 0.20 - 0.10 - 0.15) = 36 ÷ 0.55 = ¥65.45 → ¥66
```

**对比 2** - 高利润（利润 40%）：
```
建议售价 = (12 + 24) ÷ (1 - 0.20 - 0.10 - 0.40) = 36 ÷ 0.30 = ¥120
```

选择哪个取决于市场行情和销量预期。

## 五、常见问题

### Q: 为什么销价这么高？
**A**: 检查费率总和。如果佣金 + 广告 + 利润 = 60% 或更高，售价会很高。可以降低其中某些费率。

### Q: 费率总和能超过 100% 吗？
**A**: 不能。如果 佣金 + 广告 + 利润 ≥ 100%，公式分母为 0 或负数，无法计算。系统会提示错误。

### Q: 物流公式支持复杂表达式吗？
**A**: 支持。可以用 `+`, `-`, `*`, `/` 等，如 `5 + 0.08 * weight - 0.001 * weight * weight`

### Q: 新产品第一次定价有建议吗？
**A**: 建议：
1. 研究竞品定价
2. 从默认费率开始
3. 根据销量和利润调整
4. 逐步优化
